15 Melhorias Prioritárias

Impacto máximo — risco mínimo — identidade preservada
01
Border-radius nos Cards
12px em todos os .p-card, .kpi-card, .action-card. Impacto visual imediato — o maior ganho estético com menos esforço.
Visual ++++ · Esforço baixo
02
Tipografia mínima 0.52rem
Eliminar todos os textos abaixo de 0.52rem (.3rem, .34rem, .36rem, .38rem, .42rem). Crítico para uso diário 8+ horas.
Legibilidade +++++ · Médio
03
Dark Mode Unificado
3 fundos inconsistentes (#0c1f15, #122a1a, #0e2416) → 1 sistema: #0f1e16 para cards, #091510 para page.
Coerência ++++ · Médio
04
Box-shadow em Repouso
Nível 1 subtil (0 1px 3px rgba(.06)) em todos os cards. Cria profundidade visual sem peso excessivo.
Premium feel +++ · Baixo
05
Sidebar Mais Legível
Group labels: .36rem → .52rem, opacity .20 → .30. Nav items: 7px → 10px padding, .75rem → .80rem font. 30+ items diários.
UX diária ++++ · Baixo
06
CSS Variables Dark Mode
Substituir 20+ attr selectors frágeis por CSS Variables (--bg-surface, --text-primary). Sistema robusto que não quebra.
Robustez ++++ · Alto
07
Text Secondary Mais Visível
Subir opacity de .45 para .60 no texto secundário. Aparentemente pequeno, enorme impacto em leitura após 3-4 horas.
Fadiga visual +++++ · Baixo
08
Border-radius Botões/Inputs
6px em botões, 8px em inputs. Consistência com os cards. Elimina a rigidez visual dos elementos interativos.
Consistência +++ · Baixo
09
Alerts Mais Distintos
Background dos alertas: cor/04 → cor/07. Crítico vs Atenção vs Oportunidade ficam instantaneamente distintos.
Hierarquia +++ · Baixo
10
Semantic Color Tokens
Definir --success, --warning, --error, --info como CSS variables. Cores semânticas unificadas em todo o sistema.
Manutenção ++++ · Médio
11
Sidebar Mais Profunda
#0c1f15 → #060e09. Contraste sidebar vs main area aumenta. Sensação de profundidade e foco no conteúdo principal.
Depth visual ++ · Baixo
12
Pipeline Labels Legíveis
Stage names: .4rem → .60rem. Amount: .42rem mono → .75rem Cormorant. Scanning de pipeline 3x mais rápido.
Scanning ++++ · Baixo
13
Focus Rings nos Inputs
box-shadow: 0 0 0 3px rgba(28,74,53,.10) no focus. Acessibilidade keyboard. Também dá feedback visual premium.
Acessibilidade +++ · Baixo
14
Sparklines Ligeiramente Maiores
60×24px → 64×28px. Quase imperceptível em tamanho mas a linha fica mais expressiva e o dot final mais visível.
Scannability ++ · Baixo
15
Ticker Font Maior
.42rem → .54rem no market ticker. Com o scroll contínuo, texto maior reduz esforço visual do olho a seguir os dados.
Conforto visual ++ · Baixo
