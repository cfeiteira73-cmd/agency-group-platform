# Management Operating Cadence
**Owner:** Carlos Feiteira | **Version:** 2026

## Weekly Rhythm

### Monday (30 min) — Pipeline Review
**Dashboard:** `/portal` → Analytics → Revenue

Agenda:
1. Pipeline value vs. last Monday (goal: +5% MoM growth)
2. Deals advancing this week (expected CPCV/Escritura)
3. Deals at risk (L2/L3 escalation status)
4. New leads last 7 days vs. same period last month
5. Commission forecast for next 30/60/90 days

**Decision gate:** Any deal requiring Carlos intervention → scheduled for Wednesday

---

### Wednesday (20 min) — Performance + Data Quality
**Dashboard:** `/portal` → Analytics → Agent Performance

Agenda:
1. Agent activity metrics (visits, proposals, responses)
2. Data quality flags (contacts with score <60)
3. SLA violations in last 7 days
4. Platform adoption scores by agent
5. Win/loss recap from last week

**Decision gate:** Any agent below 60% platform adoption → 1:1 coaching

---

### Friday (15 min) — Weekly Close
**Agenda:**
1. Deals closed this week (commission banked)
2. CPCV deposits received
3. Upcoming escrituras next 2 weeks
4. Week reflection: what worked, what didn't
5. Priority 3 actions for next week

---

## Monthly Review (90 min, last Friday of month)

### Sections:
1. **Revenue Review** — GCI vs. target, by zone, by agent
2. **Pipeline Health** — conversion rates, avg deal size, velocity
3. **Win/Loss Analysis** — top loss reasons, win rate trend
4. **Client Experience** — NPS scores, client complaints, testimonials
5. **Platform Performance** — adoption rates, data quality avg score
6. **Agent Performance** — scorecard review, top performer recognition
7. **Market Intelligence** — zone performance, price movements, competition
8. **Next Month Forecast** — pipeline → expected revenue
